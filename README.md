# Sunrise Task Flow API

A simple Task Management REST API for the Sunrise Java course, built incrementally from the ground up.

## Requirements

- Java 25
- Maven (or use the included `./mvnw` wrapper)

## Run

```bash
./mvnw spring-boot:run
```

The server starts at `http://localhost:9999`.
